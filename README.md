# NGO Volunteer Management System

A full-stack web platform for NGOs to manage events and volunteers.

## Features
- User Authentication
- Event CRUD (Create, Read, Update, Delete)
- NGO Event Management
- Volunteer Participation

## Tech Stack
Node.js  
Express.js  
MongoDB Atlas  
EJS  
Passport.js
