# 🌍 NGO Volunteer Management System

A full-stack web application designed to help NGOs efficiently manage volunteers, organize events, and streamline participation workflows.

## 🚀 Features

* 🔐 Secure User Authentication & Authorization
* 📅 Event Management System
* ✏️ Create, Update, and Delete NGO Events
* 🤝 Volunteer Registration & Participation
* 👥 Role-based Access for NGOs and Volunteers
* 📊 Organized Dashboard for Event Tracking
* 🌐 Responsive and User-Friendly Interface

## 🛠️ Tech Stack

### Frontend

* EJS
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### Authentication

* Passport.js
